# std.regex

`std.regex` keeps regex work explicit and namespaced.

## Common helpers

- `test(pattern, subject)`
- `match(pattern, subject)`
- `findAll(pattern, subject)`
- `capture(pattern, subject)`
- `replace(pattern, subject, replacement)`
- `replaceAll(pattern, subject, replacement)`
- `split(pattern, subject)`
- `isValid(pattern)`
