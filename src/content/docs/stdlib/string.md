# std.string

`std.string` covers transforms, parsing, and helper operations on strings.

## Common helpers

- `toUpper`, `toLower`, `capitalize`
- `trim`, `trimStart`, `trimEnd`
- `split`, `join`, `lines`
- `contains`, `startsWith`, `endsWith`
- `replace`, `replaceFirst`
- `slice`, `repeat`, `reverse`
- `parseInt`, `parseFloat`
- `chars`, `bytes`, `fromChars`
- `charCode`, `fromCharCode`
