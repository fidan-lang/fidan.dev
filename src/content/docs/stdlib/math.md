# std.math

`std.math` contains the usual numeric building blocks plus a few convenience
helpers.

## Representative functions

- `sqrt`, `cbrt`, `pow`
- `sin`, `cos`, `tan`
- `log`, `log2`, `log10`, `logN`
- `floor`, `ceil`, `round`, `trunc`, `fract`
- `abs`, `min`, `max`, `clamp`
- `random`, `randomInt`
- constants like `pi`, `e`, `tau`
