# std.test

`std.test` contains assertion helpers that pair naturally with inline test
blocks and dedicated test runs.

## Common helpers

- `assert`
- `assertEq`
- `assertNe`
- `assertGt`
- `assertLt`
- `assertSome`
- `assertNothing`
- `assertType`
- `fail`
- `skip`
