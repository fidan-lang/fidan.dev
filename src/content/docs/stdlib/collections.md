# std.collections

`std.collections` contains list helpers plus a few lightweight container
shapes.

## Common helpers

- `range`
- `zip`
- `flatten`
- `enumerate`
- `chunk`
- `window`
- `partition`
- `groupBy`
- `unique`
- `reverse`
- `sort`

## Container-like helpers

- `Set`
- `Queue`
- `Stack`
