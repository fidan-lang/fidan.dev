# CLI

The main entry point is the `fidan` binary.

## Core commands

- `fidan run <file>`
- `fidan build <file>`
- `fidan check <file>`
- `fidan test <file>`
- `fidan format <file>`
- `fidan profile <file>`
- `fidan repl`
- `fidan lsp`
- `fidan explain <code>`
- `fidan explain-line <file> --line N`
- `fidan dal <subcommand>`
- `fidan toolchain <subcommand>`
