# Builtins

Fidan keeps the top-level builtin surface intentionally small.

- `print`
- `eprint`
- `input`
- `len`
- `type`
- `string`
- `integer`
- `float`
- `boolean`
- `Shared`
- `assert`
- `assert_eq`
- `assert_ne`
